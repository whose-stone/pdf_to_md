# PDF to Markdown

This repository provides tools for converting PDF documents to Markdown format. It aims to preserve formatting and maintain the integrity of the original content as much as possible.

## Installation

You can install the package using pip:

```bash
pip install pdf-to-md
```

## Usage

```bash
pdf2md yourfile.pdf
```

This command will convert `yourfile.pdf` into Markdown format and save the output in a `.md` file.
```
